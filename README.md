Курсовой проект по дисциплине "Базы данных".

Данный проект представляет собой десктопное приложение для автоматизации учета и управления деятельностью рынка. 
Приложение позволяет централизованно работать с ключевыми бизнес-процессами: управление персоналом, аренда торговых мест и складов, учет продавцов, товаров и оборудования.

База данных спроектирована и реализована в SQLite.
